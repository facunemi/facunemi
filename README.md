## Featured Work

### evidence-sanitizer

Local-first CLI tool for sanitizing sensitive evidence before sharing it or including it in security reports.

**Focus:** Python, CLI tooling, secure evidence handling, redaction rules, pentest workflows, and report preparation.

- Sanitizes sensitive tokens and credentials from evidence files.
- Designed for local-first usage.
- Built with safety, predictable behavior, and pentest documentation workflows in mind.

### Windows CTF Machine

Designed and implemented a vulnerable Windows-based CTF machine to support technical assessments for new pentesters.

**Focus:** web exploitation, exposed internal services, Windows privilege escalation, and technical assessment design.

The lab includes a realistic attack chain involving:

- Advanced web exploitation
- Exposure of internal services
- Windows privilege escalation
- Technical assessment workflow

### Alexandria

University thesis project focused on mobile application analysis from an information privacy perspective.

**Focus:** mobile application analysis, privacy, web platform development, and technical documentation.

Built a web platform to store mobile applications and their analysis results.
