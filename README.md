## Featured work

### [`evidence-sanitizer`](https://github.com/facunemi/evidence-sanitizer)

Local-first CLI for sanitizing authorized pentest evidence before it is used in reports, tickets, notes, PoCs, or internal sharing.

- Creates sanitized copies of text evidence files.
- Replaces known sensitive values with deterministic redaction markers.
- Keeps the source file untouched and avoids sending data over the network.
- Built around predictable behavior, manual review, and safer reporting workflows.

### Internal Windows CTF Lab

Designed and implemented an internal Windows-based vulnerable lab to support offensive security assessment and training scenarios.

- Focused on realistic attack-chain reasoning.
- Covered web exposure, service boundaries, credential hygiene, and Windows privilege escalation themes.
- Public details are intentionally sanitized and high-level.

Read the sanitized case study: [juannemina.com/case-studies/internal-windows-ctf-lab](https://juannemina.com/case-studies/internal-windows-ctf-lab)
